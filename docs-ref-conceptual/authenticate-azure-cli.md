> ```
>
> <span data-ttu-id="fc340-121">No PowerShell, use o cmdlet `Get-Credential`.</span><span class="sxs-lookup"><span data-stu-id="fc340-121">Under PowerShell, use the `Get-Credential` cmdlet.</span></span>

> ```
